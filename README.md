
# vue3-ts-webpack

- 基于TypeScript搭建的vue3模板，引入boostrap、elemente-plus等ui库可快速进行开发。 
- 引入axios进行二次封装的http请求方法，可以直接调用发送http请求，无需任何设置。
- 引入experssda搭建mock模拟后端响应数据，方便测试、调试。
- 引入vuex进行状态管理
- 支持对less、scss等预处理
- 支持tsx、jsx写法



## 版本
 - node: v16.13.2
 - vue: v3.2.31
 - vuex: v4.0.2
 - webpack: v5.72.0
 - typescript: v4.6.3
 - express: v4.18.1
 - bootstrap: v5.1.3

## 快速上手

```
安装： yarn install
```
```
启动： yarn run dev
```
```
打包： yarn run build
```
```
启动mock：新开一个终端输入 yarn start mock
```
